http://mde.tw/cp2020/content/HW2.html 所面臨的問題是各班採用亂數分組後, 部分學員無法利用學號登記 Github 帳號, 導致當時直接利用學號作為 Github 帳號所產生的 http://mde.tw/cp2020/content/W3.html 部分內容錯誤, 在此希望採用程式流程來產生正確的 W3 分組連結內容. 但因為分組後部分學員加退選, 因此也希望透過程式方法找出各班加選或退選的學員名單, 並將尚未納組的組員加入特定組別. 另外, 當時甲班分組後並沒有像乙班將分組結合以數列儲存, 因此必須從 W3 的網站內容中設法擷取出甲班的分組數列, 以利後續進行比對或循環建立組員連結資料之用. 總結上述, HW2 計有三項主題:

1.從 W3 網頁中擷取出甲班的分組數列

2.比對先前分組資料與最終教務處選課名單,列出事後加選或退選人員名單

3.根據組員的 Github 帳號列出正確的 W3 倉儲與網站連結
